# ЕГЭ Трекер (Steam Edition)
Расширение для Google Chrome, которое помогает отслеживать время подготовки к ЕГЭ в стиле интерфейса Steam. Следи за своим прогрессом, поддерживай стрики и мотивируй себя заниматься каждый день.

## Возможности

- **Таймер учебных сессий** — запускай и останавливай сессии одной кнопкой
- **Учёт времени** — автоматически считает суммарное количество часов обучения
- **Стрик** — отслеживает количество дней подряд, в которые ты занимался
- **Дата последнего запуска** — показывает, когда ты последний раз занимался
- **Автоматическое отслеживание сайтов** — поддерживает sdamgia.ru, fipi.ru, umsch.com (можно пополнить/удалить)
- **Автозакрытие вкладок** — опционально закрывает учебные вкладки при остановке сессии
- **Ручная настройка данных** — возможность вручную изменить время и стрик через меню настроек
- **Интерфейс в стиле Steam** — дизайн вдохновлён платформой Steam

## Установка в Chrome

1. Скачай или клонируй репозиторий:
   ```bash
   git clone https://github.com/SuperAnonymusProgramist2282009/-Tracker-Steam-Edition-.git

2. Открой Google Chrome

3. Перейди по адресу в адресной строке:
   ```bash
   chrome://extensions/

4. В правом верхнем углу страницы найди переключатель **"Режим разработчика"** и включи его

5. После включения режима разработчика появится кнопка **"Загрузить распакованное расширение"** — нажми на неё

6. В открывшемся окне выбери папку `EGE_Steam_Tracker` из скачанного репозитория

7. Расширение загрузится и появится на панели инструментов браузера. Теперь ты можешь им пользоваться!
